🌐 System Overview

Welcome to the Real-time Federated Learning Platform for Fraud Detection. This system enables organizations to collaboratively train a fraud detection model while keeping their sensitive transaction data completely private. Only model parameters are shared, never raw data.

⚙️ How It Works

  1. 1. Register Your Organization
    Create an account and register as a client with your available transaction data size.
  2. 2. Upload Your Local Data
    Your fraud detection dataset stays on your secure system (never transmitted).
  3. 3. Participate in Training Rounds
    Download the global model, train locally on your data, and upload trained parameters.
  4. 4. Server Aggregates
    Our server uses FedAvg (Federated Averaging) to combine all client models securely.
  5. 5. Repeat & Improve
    Continue participating in rounds. Model performance improves each round.
